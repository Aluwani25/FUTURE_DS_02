# Customer Churn & Retention Analysis
This project focuses on analyzing customer churn behavior in a telecommunications company. The goal is to identify patterns, key drivers of churn, and provide actionable insights to improve customer retention.
The analysis simulates a real-world business scenario where data is used to support strategic decision-making and reduce customer loss.

# Objectives
- Identify factors contributing to customer churn
- Analyze customer behavior based on tenure, services, and billing
- Measure churn rate and retention performance
- Provide data-driven recommendations to improve customer retention

# Tools & Technologies
- Python (Pandas) → Data cleaning and preprocessing
- Power BI → Interactive dashboard and visualization
- CSV / Excel → Data storage

# Key Metrics (KPIs)
- Total Customers
- Churned Customers
- Churn Rate (%)
- Total Revenue

# Analysis & Visualizations
# Churn Overview
- Distribution of churned vs retained customers
- Overall churn rate

# Churn by Contract Type
- Identifies which contract types have the highest churn
- Month-to-month contracts show significantly higher churn

# Churn by Tenure
- New customers are more likely to churn
- Retention improves as tenure increases

# Charges vs Churn
- Customers with higher monthly charges tend to churn more

# Churn by Services
- Internet service type impacts churn behavior

# Key Insights
- Introduce incentives for long-term contracts
- Improve onboarding experience for new customers
- Offer pricing strategies or bundles for high-charge customers
- Focus retention campaigns on high-risk segments

# Conclusion
This analysis demonstrates how data can be used to understand customer behavior and reduce churn. By focusing on high-risk segments and improving retention strategies, businesses can increase customer lifetime value and overall profitability.
